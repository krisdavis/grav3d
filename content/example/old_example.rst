.. _example:


Examples
========

Here, the program libraries for Grav3D octree will be used to:

    - design a tensor mesh
    - create a density contrast model
    - forward model the gravity anomaly
    - invert gravity anomaly data to recover a density contrast model

Zip folders containing all necessary files can be downloaded here:

    .. - `Download and open the zip folder containing the entire Grav3D example <https://github.com/ubcgif/Grav3D/raw/master/assets/grav3d_example.zip>`__

The full examples are parse into 5 sections:

.. toctree::
    :maxdepth: 2

    Design tensor mesh <example/create_mesh>
    Create tensor model <example/create_model>
    Forward modeling <example/fwd>
    Generate sensitivity weights <example/weights>
    Inversion <example/inv>
